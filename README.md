# Real-Time Collaborative Editor

This project is a real-time collaborative text editor similar to Google Docs.

## Technologies Used
- React.js
- Node.js
- Express.js
- Socket.IO

## Features
- Real-time text synchronization
- Multiple users can edit simultaneously
- Instant updates across connected clients

## Project Structure
collab-editor
 ├── backend
 │   └── server.js
 └── frontend
     └── React application

## How to Run

### Backend
cd backend
npm install
node server.js

### Frontend
cd frontend
npm install
npm start

Open http://localhost:3000
